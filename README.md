C3 Steam Export Guide
Over the years there have been multiple issues when exporting you Construct 3 game to Steam. This guide aims to help new users understand the current issues that can happen when doing so.

https://andreyin.github.io/c3_steam_export_guide/steam-export/

Running the site locally
This requires the following installed on your system:

node.js
To test the website locally, clone the source code:

git clone https://github.com/andreyin/c3_steam_export_guide --recurse-submodules
Then simply run the following commands:

npm ci
npm run docs:dev
The website should now be running on localhost:5173 (or a port shown on the terminal).
